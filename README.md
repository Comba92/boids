# Boids - Flocking simulation in Lua Love2D
Love2D implementation of the Boids simulation.
https://en.wikipedia.org/wiki/Boids

![preview](preview.gif)

## Build
The project requires [Love2D](https://love2d.org/).
Simply run:
```
love .
```
on the project's root.
